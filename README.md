# 🎬 MovieCircle

MovieCircle is a private movie-tracking and recommendation platform built for friend groups.

Instead of individually recommending movies to everyone, users can create private circles, add movies using IMDb links, rate them, mark them as watched, and discover what the group loves most.

---

# ✨ Features

## 👥 Private Groups

* Create private movie circles
* Join groups using invite codes
* Manage movies together with friends

## 🎬 Add Movies via IMDb

* Paste IMDb links
* Movie details automatically fetched from TMDB
* Posters, ratings, genres, runtime, and more

## ⭐ Ratings & Recommendations

* Group members can rate movies
* Top-rated movies shown automatically
* Personalized recommendations based on unwatched movies

## ✅ Watch Tracking

* Mark movies as watched
* Track who watched what
* Watch history with dates

## 🎯 Filters & Sorting

* Genre filter
* Language filter
* Country-wise sorting
* Minimum rating filter

## 🌙 Cinematic UI

* Dark modern interface
* Smooth animations
* Optimized for desktop
* Mobile responsive version ongoing
